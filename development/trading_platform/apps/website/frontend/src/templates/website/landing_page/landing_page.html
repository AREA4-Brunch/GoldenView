Frequently asked questions
Everything you need to know about the application.
Why GoldenView and not other applications?
We have identified a significant issue common to all of them - non user-friendly interface which results in a steep learning curve for beginners. The “GoldenView” would resolve this issue with its intuitive interface while preserving some of the most significant features in existing programs. This would enable users with little to no prior experience to enter the world of finance with ease.
What features do I get?
You get a lot of features such as: best UX, asset values in real time, broker help, access from any device and many more.
Does using GoldenView pose any risks in trading?
Any trades done on any platform always pose a risk, since the stock market is changing in unpredictable ways. Please understand that by using GoldenView you acknowledge that we are not at fault if you have any losses.
How do I become a Broker for GoldenView?
Sending a broker request is very easy. You just need to press the Become a Broker button in the top menu and fill in the form. Wait for the request to be accepted and you are ready to help others.
Are their any costs for using GoldenView?
GoldenView is currently a free non-profit software that is made as a project for the School of Electrical Engineering, University of Belgrade.
